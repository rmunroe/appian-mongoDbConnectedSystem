This plugin implements an Appian `Connected System <https://docs.appian.com/suite/help/latest/Connected_System.html>`_ for `MongoDB <https://www.mongodb.com>`_ using the `MongoDB Java Driver <https://mongodb.github.io/mongo-java-driver/3.12/>`_ in synchronous mode. It provides Appian with a low-code integration to MongoDB without the developer (or *designer* in Appian parlance) needing to understand or work with the MongoDB REST API. It also includes many conveniences such as date/time and MongoDB custom type handling (ObjectIds, Binary, etc.), as well as a full suite of :doc:`expressions` to construct `MongoDB Query Documents <https://docs.mongodb.com/manual/tutorial/query-documents/>`_ (their JSON-based query language) via Appian expressions, instead of tedious, manual string building.

The official plugin JAR distribution location is `this entry in the Appian Community App Market <https://community.appian.com/b/appmarket/posts/mongodb-connected-system>`_.
